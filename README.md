# Kuromi Midnight

A violet Discord theme, calm atmospheric design.

Designed for users who prefer a minimal, moody interface with subtle glow accents and smooth visual depth — without overwhelming neon styling.

---

## Installation

### Vencord / BetterDiscord

1. Install **Vencord** or **BetterDiscord**
2. Download the latest release:
   - `Kuromi-Midnight-v1.0.0.css`
3. Move the file into your themes folder:
   - Vencord: Themes directory
   - BetterDiscord: themes folder
4. Enable **Kuromi Midnight** in settings

---

## Features

- Violet + midnight gradient system
- Glassmorphism UI layering
- Soft Kuromi-inspired accent system
- Subtle glow interactions
- Rounded “plush UI” design language
- Improved chat readability styling
- Floating capsule-style input bar
- Enhanced profile popouts
- Smooth hover animations across UI
- Custom scrollbar design
